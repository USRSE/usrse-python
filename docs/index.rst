=============
US-RSE Python
=============

.. image:: https://img.shields.io/github/stars/USRSE/usrse-python?style=social
    :alt: GitHub stars
    :target: https://github.com/USRSE/usrse-python/stargazers


US-RSE Python is a simple client that lets you programatically interact with
US-RSE content, including events, DEI content, posts, and newsletters. The
client current allows for viewing of content in tables or download to json,
and we have more interactions coming soon!

To see the code, head over to the `repository <https://github.com/USRSE/usrse-python/>`_.
To browse modules available as containers, see `the library <https://us-rse.org/usrse-python/>`_.

.. _main-getting-started:

----------------------------------
Getting started with US-RSE Python
----------------------------------

US-RSE Python can be installed from pypi or directly from the repository. See :ref:`getting_started-installation` for
installation, and then the :ref:`getting-started` section for using the client.
